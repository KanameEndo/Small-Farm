# 実行手順
$ git clone git@github.com:KanameEndo/Small-Farm.git</br>
$ cd Small-Farm</br>
$ bundle install</br>
$ yarn install</br>
$ rails db:create</br>
$ rails db:migrate</br>
$ rails db:seed</br>
$ rails s</br>
