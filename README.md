# README

# 開発言語
- Ruby 3.0.1
- Rails 6.0.5

# 就業Termの技術
- device
- お気に入り機能

# カリキュラム外の技術
- ransack

# 実行手順
$ git clone git@github.com:KanameEndo/Small-Farm.git
$ cd Small-Farm
$ bundle install
$ yarn install
$ rails db:create
$ rails db:migrate
$ rails db:seed
$ rails s
